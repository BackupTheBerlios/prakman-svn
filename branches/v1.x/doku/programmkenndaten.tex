\section{Lösungsverfahren und Algorithmen}
\label{sec:Algorithmen}
Es werden keine außergewöhnlichen Algorithmen verwendet.

\section{Hardware- und Softwareanforderungen}
\label{sec:Anforderungen}
Zum Betrieb des Programms erforderlich sind
\begin{itemize}
\item Java Version 1.5 oder höher
\item Für Mehrbenutzer-Verwendung: Eine externe Datenbank (Postgres)
\item Bei lokaler Nutzung: Speicherplatz für die zu haltenden Daten
\end{itemize}

\section{Benötigte Dateien und Directories}
\label{sec:DateienUndDirectories}
Zum Start des Programms wird allein die Datei ``prakman.jar'' benötigt, welche das Programm sowie alle Treiber für die unterstützten Datenbanken enthält.
